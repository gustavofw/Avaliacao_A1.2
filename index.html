Dog API

The internet's biggest collection of open source dog pictures.

Documentação

A Dog API (API de cachorros) em "https://dog.ceo/dog-api/" é uma API pública que fornece acesso a imagens aleatórias de cachorros de diferentes raças, bem como informações sobre raças específicas. Ela é usada principalmente para desenvolver aplicativos e sites relacionados a cachorros, como galerias de fotos de cachorros, aplicativos de informações sobre raças, jogos de adivinhação de raças, entre outros. A API possui vários endpoints que permitem acessar diferentes tipos de informações e recursos relacionados a cachorros. Eis alguns dos principais recursos que você pode utilizar com esta API:

  • Endpoint para listagem de raças: URL: "https://dog.ceo/api/breeds/list/all" Esta endpoint retorna uma lista de todas as raças de cães disponíveis, juntamente com suas sub-raças, se houver.
  • Endpoint para imagens aleatórias de cachorros: URL: "https://dog.ceo/api/breeds/image/random" Esse endpoint retorna uma URL de uma imagem de cachorro aleatória.
  • Endpoint para imagens por raça de cachorro: URL: "https://dog.ceo/api/breed/{raça}/images" Esse endpoint retorna um conjunto de todas as imagens de uma raça, por exemplo, cão de caça.
  • Endpoint para imagens por sub-raça de cachorro: URL: "https://dog.ceo/api/breed/{raça}/list" Retorna um array de todas as sub-raças de uma raça.
  • Endpoint para navegar na lista de raças: URL: "https://dog.ceo/api/breed/{raça}/images/random" Retorna navegação entre as raças da API.
